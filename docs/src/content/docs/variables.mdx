# CSS Variables

Varvara CSS is customizable using [CSS Variables](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties) (also called CSS custom properties).

## Border

<Code
  code={`--va-border-width: 2px;
--va-border-color: var(--va-text-normal);`}
  lang="css"
/>

## Colors

<Code
  code={`--va-color-primary: #2a303c;
--va-background-primary: #b3ccd6;
--va-text-normal: #2a303c;
--va-text-weak: color-mix(in srgb, var(--va-text-normal) 20%, transparent);`}
  lang="css"
/>
